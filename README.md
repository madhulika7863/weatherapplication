🌦️ Weather App

A responsive and user-friendly Weather Application built using HTML, CSS, and JavaScript. This app fetches real-time weather data using the OpenWeather API and displays accurate weather information for any city worldwide.


## 🚀 Features
🌍 Search weather by city name
🌡️ Real-time temperature display
☁️ Weather conditions (Clear, Clouds, Rain, etc.)
💨 Wind speed and humidity
📱 Fully responsive design (mobile + desktop)
⚡ Fast and lightweight UI


## 🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
API: OpenWeather API
Data Format: JSON
📂 Project Structure
Weather-App/
│
├── weatherapp.css
├── weatherapp.html
└── README.md

### How It Works
- User enters a city name
- App sends a request to OpenWeather API
- API returns weather data in JSON format
- Data is parsed and displayed on UI
